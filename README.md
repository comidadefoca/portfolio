# Eduardo Leandro — Product Designer Portfolio

A premium, editorial-inspired digital portfolio featuring a multi-language system, interactive case studies, and a high-performance minimalist design.

[Live Demo →](https://comidadefoca.github.io/portfolio)

---

## ✨ Key Features

### 🌍 Multi-Language System (PT/EN)
- **Auto-Detection:** Detects browser language on first visit.
- **Persistence:** Remembers user preference across sessions via `localStorage`.
- **Zero-Flash Switching:** Instant language toggling without page reloads.
- **Dynamic Meta Tags:** Browser tab titles update dynamically based on the active language.

### 📖 Interactive Case Studies
- **Progress Wizard:** A sticky, scroll-aware navigation system for long-form case studies.
- **Visual Storytelling:** Editorial grid layout with deep attention to typography (Inter) and spacing.
- **Responsive Fluidity:** Optimized for mobile, tablet, and desktop viewing.

### 🎨 Editorial Aesthetic
- **Stone Palette:** A curated minimalist color system for a premium, grounded feel.
- **Micro-interactions:** Subtle scroll animations and hover effects that enhance engagement without distraction.
- **Performance First:** Built with vanilla technologies for lightning-fast load times.

---

## 🛠 Tech Stack

- **HTML5:** Semantic structure for accessibility and SEO.
- **Tailwind CSS:** Modern utility-first styling for consistent design systems.
- **Vanilla JavaScript:** High-performance, lightweight logic for animations and interactivity.
- **Iconify:** Scalable vector icons for a crisp look.

## 🚀 Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/comidadefoca/portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd portfolio
   ```

3. **Run a local server:**
   You can use any light server (VSCode Live Server, Python's `http.server`, etc.):
   ```bash
   python3 -m http.server
   ```

4. **View in browser:**
   Open `http://localhost:8000`
